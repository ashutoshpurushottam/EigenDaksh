UltraCash Merchant

The merchant variant of the UltraCash application currently installed at over 3000 merchant outlets in Bengaluru, India. The application lets merchants accept online/offline payments from the customer variant. The application dynamically updates all online transactions for the merchant and also lets the outlets receive transaction reports on email. Merchants can also share payment links with their customers using which customers can make payments.

The unique feature of the application is that it supports offline payment using sound waves from the customer variant. The app sends the details to the customer variant through sound waves and also pings the server at the backend about the transaction. After the transaction is completed, the merchant application posts transaction details to the backend.

The application has been developed using the latest technologies of the Android development domain like RxAndroid, Dependency Injection, and Room database. It has not reported any crash in the production for over a year.
