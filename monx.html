Mon-X

Mon-X creates a market where users could exchange money directly without paying any fees to third parties. This way, sellers and buyers of the currency would be able to save money on the transaction.

The application Mon-X has many functionalities like remote notifications, internationalization, uses API to get foreign currency prices, animation, authentication, core data, Firebase database and Firebase storage.

The screen on the left shows the main screen; users can search for the currency exchange rate. The users then would submit the amount to exchange, and the total in the new currency would then display. The user would have the ability to edit the amounts and create an offer. The second screen shows the ability to communicate through the app chat with a person interested in the exchange. The last picture to the right shows the possibility to find how far are the users posting their offers and information relevant to help you decide if you want to engage in an exchange transaction with that user.
