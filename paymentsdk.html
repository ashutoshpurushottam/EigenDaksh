UltraCash Payment SDK

Android SDK which can be integrated into any Android application to support payment through a variety of options including Credit Card, Debit Card, Bank Account, UPI Account or Ultracash Wallet. The user can configure these accounts in the application through this SDK or make payment through any of the configured accounts.

It is a robust application which fulfills all compliance requirements of Payment Card Industry Data Security Standard and National Payments Corporation of India. The SDK has been developed using the latest technologies of the Android development domain like RxAndroid, Dependency Injection, and Room database.
